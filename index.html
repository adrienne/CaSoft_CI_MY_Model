    CaSoft_CI_MY_Model

    This is a generic Model for CodeIgniter projects.

    This Model has a collection of default methods to save and retrieve data from database.

    Depends

    • Works with CodeIgniter 2.x
    • Last tested with CodeIgniter 2.1.0.
